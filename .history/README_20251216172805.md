# Bruna Guarizo Portfolio Website

Portfolio website for brand identity and package designer, built with React, JSX, and CSS Modules.

## Technologies

- React 18
- Vite
- JSX
- CSS Modules
- Motion (Framer Motion) - for animations
- Simplex Noise - for wavy background effects

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The build will be generated in the `dist/` folder.

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```
src/
  components/
    About/          # About section component
    Button/         # Reusable button with hover gradient effect
    FAQ/            # FAQ section component
    Footer/         # Footer component
    HamburgerMenu/  # Mobile navigation menu
    Header/         # Navigation header
    Process/        # Process section component
    Projects/       # Projects showcase
    Services/       # Services section
    Testimonials/   # Testimonials section
    WavyBackground/ # Animated wavy background component
  pages/
    Home/           # Homepage with hero section
  assets/
    images.js       # Image URLs configuration
  App.jsx          # Main application component
  index.css        # Global styles and CSS variables
  main.jsx         # Application entry point
```

## Features

- **Dark Theme**: Consistent dark color scheme using CSS variables
- **Animated Background**: Wavy background effect using canvas and simplex noise
- **Responsive Design**: Mobile-first approach with breakpoints
- **Smooth Animations**: Button hover effects with gradient borders
- **Component-based Architecture**: Reusable components with CSS Modules

## CSS Variables

The project uses CSS variables defined in `src/index.css` for consistent styling:

- **Colors**: `--white`, `--black`, `--gray-light`, `--gray-medium`, `--gray-dark`, `--gray-very-dark`
- **Typography**: `--font-size-small`, `--font-size-medium`, `--font-size-regular`, `--font-size-large`
- **Spacing**: `--spacing-xs` through `--spacing-5xl`
- **Border Radius**: `--radius-sm`, `--radius-md`, `--radius-lg`
- **Transitions**: `--transition-fast`, `--transition-normal`
