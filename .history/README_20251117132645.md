# Branding Portfolio Website

Um website de portfólio para designer de identidade de marca e embalagens, criado com React, JSX e CSS Modules.

## Tecnologias Utilizadas

- React 18
- Vite
- JSX
- CSS Modules

## Como Executar

1. Instale as dependências:
```bash
npm install
```

2. Execute o servidor de desenvolvimento:
```bash
npm run dev
```

3. Abra o navegador em `http://localhost:5173`

## Configurando as Imagens e Backgrounds

O projeto está configurado para usar as imagens e backgrounds do site original do Framer. 

### Para Imagens:
1. **Consulte o arquivo `INSTRUCOES_IMAGENS.md`** para instruções detalhadas
2. Abra o site original: https://right-tenure-137382.framer.app/
3. Use o console do navegador para extrair as URLs das imagens
4. Cole as URLs no arquivo `src/assets/images.js`

### Para Backgrounds Animados:
1. **Consulte o arquivo `INSTRUCOES_BACKGROUNDS.md`** para instruções detalhadas
2. Use o script no console do navegador para extrair backgrounds com animações
3. Cole a URL do background em `src/assets/images.js` no campo `heroBackground`
4. O componente `AnimatedBackground` aplicará automaticamente o efeito de movimento

**Alternativa:** Se preferir usar imagens locais:
1. Baixe as imagens do site original
2. Coloque-as na pasta `public/images/`
3. Atualize `src/assets/images.js` para usar caminhos locais: `/images/nome-da-imagem.jpg`

## Esquema de Cores

O site usa um esquema de cores **dark mode**:
- Fundo: Preto (#000)
- Texto: Branco (#fff)
- Elementos secundários: Cinza claro (#ccc, #999)
- Cards: Cinza escuro (#111) com bordas (#333)
